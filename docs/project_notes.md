Компоненты:
* Микроконтроллер (MCU) STM32H743IIT в корпусе LQFP176(24x24 мм).  - "мозг" системы. Будет читать данные с датчиков, производить расчеты и управлять моторами.

Инерциальные датчики ICM20602, ICM42688P - измеряют ускорение и угловую скорость. Нужны для определения ориентации и движения аппарата.

* Барометры BMP280(2 шт) - Измеряют атмосферное давление для определения высоты. Два для резервирования (на случай отказа одного).

EEPROM CAT24C256 - Микросхема памяти. Хранит настройки, калибровочные данные, которые не должны стираться при выключении питания.

Магнитометр QMC5883L - Определяет направление в пространстве (цифровой компас).

Драйверы интерфейсов: SN65HVD232 (для CAN), [драйвер для RS485] - Преобразуют логические уровни MCU (0/3.3В) в уровни промышленных интерфейсов, устойчивых к помехам.

Буфер 74HC125D - "Усилитель" для сигналов ШИМ. Гарантирует, что сигнал дойдет до моторов без искажений.

Стабилизатор питания ST1S14PHR - Преобразует входное напряжение (например, 12V) в стабильные 3.3V для питания MCU и всей логики.



Удаляю пункты после выполнения:
    • Подключить по SPI два инерциальных датчика - ICM20602 и ICM42688P. Нужно подключить линии SPI (использовать по своей линии spi на каждый датчик), а также подключить линии INT на любые GPIO.
    • 2шт барометры (для резервирования) BMP280, на разные шины I2C;
    • подключить к микроконтроллеру CAT24C256 по I2C;
    • магнитометр 1 шт (QMC5883L); 
    • управление 8 моторов. Наружу выводятся каналы ШИМ, двух счетчиков-таймеров (ножки обозначены TIMx, например TIM2_CH1 – таймер 2 канал 1. Согласовать номер таймера со мной. Линии проводятся через буферы 74HC125D.
    • Установить на плате джампер BOOT0 микроконтроллера – когда не замкнут на выводе 0, когда замкнут – 3.3В.
    • К микроконтроллеру подключить кварц KSE-7U16000M9Z143ZA3 или подобный, на 16МГц.
    • Подключить к микроконтроллеру держатель SD карты 112J-TDAR-R01, для нее есть специальные выводы, схема гуглится.
    • На любые GPIO подключить 5 светодиодов через резисторы 5.1к.
    • Выводим наружу разъем DS1079-F500BS, все линии наружу идут через него.
    • Электропитание сделать на микросхеме ST1S14PHR по типовой схеме включения;
    • Вывести наружу 3 UART, 2 CAN интерфейса через драйвер SN65HVD232 и 2 RS485 через драйвер .
